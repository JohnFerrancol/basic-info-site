## Overview

This is a project from [The Odin Project](https://theodinproject.com): [Project: Basic Information Site](https://www.theodinproject.com/lessons/nodejs-basic-informational-site). This project is a very basic site to create a vanilla Node JS server, and use the fs module in Node to read files

## Learning Points

- Understanding how to use Node to run JavaScript files outside of the browser
- Using the default node packages like http and fs to create a node server that reads and renders HTML files based on different routes

## Getting Started

### Prerequisites

You will need to install the latest version of npm to get started on using this project

- npm

```sh
npm install npm@latest -g
```

### Installation

Getting started on using the basic information site

1. Cloning the repository

```sh
git clone git@github.com:JohnFerrancol/basic-info-site.git
```

2. Running the JavaScript files

```sh
node index.js
```

4. Open in web browser via: http://localhost:8080
